Getting Started

Plex Requests is relatively straight forward to get installed. It has a single requirement, and there are instructions for the Mac, Windows, and Linux. In addition there are some configuration details available, including running the application on start-up. If there are any issues you would like to improve the docs please open a pull request over on GitHub!

Requirements

Plex Requests is built upon Hapi. It's a Node platform akin to ExpressJS that is great for creating robust applications. The only requirement is Node which runs on Windows, Mac OS X, and Linux.

The application requires at least v4 of Node due to a few dependencies. It has also been tested with v5 if you want to be on the cutting edge. You can download Node from their downloads page if you don't have it already installed.

Installation

Mac OS X

Installation is fairly straightforward. You can get the app one of two ways: using Git, or download release specific archive. Releases can be found on GitHub on the releases page. If you plan on using Git and you don't have it installed head over there for instructions.

Once git is installed, simply open up a terminal window and navigate to where you'd like to install Plex Requests. The below command will get you a fresh and up to date version of the app.

git clone https://github.com/lokenx/plex-requests.git

Once that is done, open up the newly created folder and tell NPM to install everything you need. If your terminal says something about NPM being an unrecognized command, make sure you have installed Node as mentioned up top.

cd plex-requests
npm install

After a few minutes of npm installing things you are good to go! Simply type the below command to start things up. Things should now be accessible at localhost:8000.

npm start
